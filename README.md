# Nuxt 4 Boilerplate Template

Modern, high-performance boilerplate menggunakan **Nuxt 4**, **Nuxt UI v3**, **Pinia 3**, dan **TypeScript**.

## 🚀 Fitur Utama
- **Nuxt 4**: Struktur direktori modern (`app/`) dan efisiensi tinggi.
- **Nuxt UI v3**: Komponen UI premium berbasis Tailwind CSS v4 & Headless UI.
- **Pinia 3**: State management yang type-safe dan intuitif.
- **TypeScript**: Dukungan penuh TypeScript untuk development yang lebih aman.
- **Dark Mode**: Terintegrasi langsung dengan Nuxt UI.

## 🛠 Panduan Instalasi & Penggunaan

### 1. Prasyarat
Pastikan Anda memiliki Node.js terinstal (direkomendasikan versi terbaru).

### 2. Instalasi Dependensi
Jalankan perintah berikut di root project:
```bash
npm install
```

### 3. Development Server
Jalankan server pengembangan:
```bash
npm run dev
```
Akses di `http://localhost:3000`.

### 4. Build untuk Production
```bash
npm run build
```

## 📂 Struktur Folder (Nuxt 4)
- `app/`: Direktori utama aplikasi (sesuai standar Nuxt 4).
  - `layouts/`: Template layout aplikasi.
  - `pages/`: Halaman/route aplikasi.
  - `stores/`: State management (Pinia).
  - `app.vue`: Entry point utama aplikasi.
- `public/`: Asset statis (images, favicon, dll).
- `nuxt.config.ts`: Konfigurasi Nuxt.

## 📦 Terinstal dalam Boilerplate:
- `nuxt@4.x`
- `@nuxt/ui@^4.0.0` (Tailwind CSS included)
- `@pinia/nuxt`
- `pinia`
- `typescript`
